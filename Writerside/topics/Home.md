# TaskLynx / Nest Hotel – Docs

![TaskLynx-logo-oval.svg](TaskLynx-logo-oval.svg)

## Introducción

TaskLynx es un ecosistema compuesto por TaskLynx Api, TaskLynx Business y TaskLynx Mobile.  
TaskLynx Api es una API REST que permite la comunicación entre TaskLynx Business y TaskLynx Mobile.  

NestHotel es una API REST que permite la comunicación con TaskLynx Business y así poder gestionar 
las limpiezas de las habitaciones de un hotel.
